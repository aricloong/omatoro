# 🍣 omatoro.com

> 寿司百科 · 旬味 · 清酒配对

**omatoro** 是一个由 omatoro 匠心团队打造的寿司社区。

## 功能
- 📖 30种寿司鱼类图鉴（产地、季节、烹饪方式）
- 🌿 旬味日历（每月最佳时令鱼类）
- 🍶 清酒配对计算器
- 🔥 热量计算器
- 🎯 寿司知识问答
- ✏️ 用户投稿 & 品鉴笔记
- 🌓 深色/浅色主题切换
- 🌍 多语言支持（中/日/英/法/西/阿/俄）

## 技术栈
- [Astro](https://astro.build/) — 静态站点生成
- Cloudflare Pages — 部署
- 本地化数据驱动（无需数据库）

## 本地开发

```bash
npm install
npm run dev
```

## 部署

推送到 `main` 分支，Cloudflare Pages 自动触发部署。
